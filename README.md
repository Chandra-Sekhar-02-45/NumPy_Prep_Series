# NumPy Prep Series

Welcome to the **NumPy Prep Series**! This repository documents my journey of mastering **NumPy**, the fundamental package for scientific computing in Python.

Here, you will find detailed notes, code examples, and practical implementations covering everything from basic array creation to advanced data manipulation.

## Course Curriculum

This series covers the following key topics, structured to take you from a beginner to an advanced user:

### 1. Introduction & Basics
*   What is NumPy and why use it?
*   NumPy Arrays (`ndarray`) vs. Python Lists.
*   Performance comparison (Speed & Memory).
*   Understanding Vectorization and Contiguous Memory.

### 2. Working with Arrays
*   **Creating Arrays**: `np.array()`, `np.arange()`, `np.zeros()`, `np.ones()`, `np.linspace()`, `np.empty()`.
*   **Attributes**: Shape, Size, Dtype, Ndim.
*   **Indexing & Slicing**:
    *   Basic indexing.
    *   Slicing 1D and Multidimensional arrays.
    *   Boolean indexing and Conditional selection.

### 3. Array Operations
*   **Arithmetic**: Element-wise addition, subtraction, multiplication, division.
*   **Broadcasting**: Rules and examples.
*   **Mathematical Functions**: `np.sqrt()`, `np.square()`, `np.abs()`, `np.power()`.

### 4. Data Manipulation
*   **Reshaping**: `reshape()`, `ravel()`, `flatten()`.
*   **Transposing**: `T`, `transpose()`.
*   **Joining**: `np.concatenate()`, `np.vstack()`, `np.hstack()`, `np.append()`.
*   **Splitting**: Breaking arrays into sub-arrays.

### 5. Statistical & Aggregation Functions
*   **Aggregations**: `sum()`, `min()`, `max()`, `mean()`, `median()`, `std()`, `var()`.
*   **Cumulative**: `cumsum()`, `cumprod()`.
*   **Sorting**: `sort()`, `argsort()`.
*   **Searching**: `where()`, `argmax()`, `argmin()`.

### 6. Random Number Generation
*   Generating random integers and floats.
*   Sampling from distributions (Uniform, Normal).
*   Random seeds for reproducibility.

## Projects & Real-World Applications
This repository will also include implementations of real-world problems to demonstrate the power of NumPy:
*   **Data Preprocessing**: Cleaning and preparing datasets for Machine Learning.
*   **Image Processing**: Manipulating images as pixel arrays.
*   **Statistical Analysis**: Analyzing datasets using statistical functions.
*   **Linear Algebra**: Solving systems of equations and matrix operations.
